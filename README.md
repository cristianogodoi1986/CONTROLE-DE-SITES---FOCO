# 🧠 Focus Control Script (Windows)

Script automatizado para **bloqueio e gerenciamento de sites distrativos** diretamente no Windows, sem necessidade de softwares adicionais.

---

## 🚀 Visão geral

Este projeto foi criado com um objetivo simples:

> **Aumentar o foco eliminando distrações no nível do sistema**

Ao invés de depender de extensões ou força de vontade, o script atua diretamente no arquivo `hosts` do Windows, bloqueando o acesso a sites indesejados em todos os navegadores.

---

## 🎯 Funcionalidades

- 🔒 Bloqueio de sites distrativos
- 🔓 Desbloqueio rápido (1 clique)
- 📋 Verificação de bloqueios ativos
- 💾 Backup automático do arquivo `hosts`
- ⚡ Execução leve e rápida
- 🌐 Funciona em qualquer navegador

---

## 🛠️ Tecnologias utilizadas

- Batch Script (.bat)
- Arquivo de sistema `hosts`
- Comandos nativos do Windows

## ⚠️ Observações importantes

- Necessário executar como administrador
- Funciona no nível do sistema (todos os navegadores)
- Pode ser contornado com VPN ou proxy
- Alterações são reversíveis

---

## 🧠 Por que usar isso?

Hoje, o maior problema de produtividade não é falta de capacidade.

É excesso de distração.

Este script permite:

- Criar um ambiente mais controlado
- Reduzir interrupções
- Aumentar foco em estudos ou trabalho
- Padronizar uso em ambientes corporativos

## 📊 Casos de uso

- Estudantes em preparação para provas
